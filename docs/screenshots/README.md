# Screenshots

Place dashboard screenshots in this folder. The README references them via:

```markdown
![Dashboard](docs/screenshots/01-intel.png)
```

## Recommended captures

1. **01-intel.png** — F1 INTEL tab showing morning brief, top opportunities, alerts
2. **02-scanner.png** — F2 SCANNER showing market table with edges and signals
3. **03-analytics.png** — F3 ANALYTICS showing correlation heatmap
4. **04-execute.png** — F4 EXECUTE showing trade blotter and live P&L
5. **05-risk.png** — F5 RISK showing P&L calendar and equity curve
6. **06-review.png** — F6 REVIEW showing trade journal and RL lessons

## Capture guide

- Use `Win+Shift+S` (Windows) or `Cmd+Shift+4` (Mac) for selection capture
- Resolution: 1920x1080 or higher
- Format: PNG (better text rendering than JPG)
- Crop to just the dashboard area (no browser chrome)
- Keep file size under 500KB each (optimize with TinyPNG if needed)
